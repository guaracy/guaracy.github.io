Blog minimal

Enviado em Mar 9, 2017
#blog-minimal #emacs #org-mode #w3.css

Não comentei antes pois está no GitHub há uma semana e vi no blog da Sacha no meio desta semana. Mas o Blog minimal é uma opção interessante para quem usa Emacs e deseja um blog estático. Um resumo:

  • No início é necessário configurar algumas variáveis o que não deve significar problema para um usuário do emacs.
  • Para os artigos ele utiliza o org-mode que é uma opção que pode escrever um comentário, um artigo científico ou um livro. Certamente adequado para qualquer situação.
  • Usa o mustache para os templates das páginas html.
  • Como foi desenvolvido por um estudante chinês, a apresentação do blog e as imagens não fecham com o meu gosto. Seria necessário utilizar um outro css e imagens. Acho que vou fazer alguns testes com o w3.css que acho interessante.

Ainda não temos uma documentação e eu ainda não analisei o fonte para ter maiores informações (categorias, tags, etc.).
